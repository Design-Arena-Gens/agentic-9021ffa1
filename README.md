## KK Workspace

KK Workspace is a polished marketing site for a fictional knowledge operating system that keeps product, research, and delivery teams aligned. The single-page experience showcases the product narrative, highlights workflow benefits, and captures leads through an in-page form.

### Tech Stack

- Next.js App Router (TypeScript)
- Tailwind CSS v4
- Responsive design with gradients, glassmorphism, and reusable data-driven sections

### Local Development

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to explore the experience. The main entry point is `src/app/page.tsx`.

### Production Build

```bash
npm run build
npm start
```

### Deploy

The project is ready for seamless deployments to Vercel using `vercel deploy`.
